# Aria2-App
A standalone Aria2 app

https://github.com/maxogden/electron-packager

# Create the app

## Mac OS X

```sh
electron-packager . Aria2 --platform=darwin --arch=x64 --overwrite --ignore="node_modules/(electron-packager|electron-prebuilt)" --icon=icon/icon.icns
```
